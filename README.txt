--- STEPS TO FOLLOW ---

1. Extract the zip file/ copy the folder to the server path (localhost/ webserver)
2. Run 'composer install' command in the terminal to install all dependencies
3. Go to localhost/pass-sorter in the browser
4. For Postman use the URL : 'localhost/pass-sorter/api/sort'
5. To check the working example please visit https://sabinchacko.com/pass-sorter/

--- Complexity of the Algorithm ---
The Algorithm is having complexity of O(n)
